 Kim's Configurations Collection.
-----------

## Usage

    ./konfig <conf|exec|insert|backup>

#### Execution.

    ./konfig exec [all | <script list>]

#### Configuration.

    ./konfig conf [all | <config list>]

#### Insert new config.

    ./konfig insert <config> <local path> <filename>

#### Backup configs.

    ./konfig backup [all |<config list>]
